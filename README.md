# 🅰️ Angular Practice

A practice repository dedicated to exploring and learning the Angular framework. This project serves as a sandbox for implementing various Angular features, standalone components, and modern reactive patterns like Angular Signals.

## 🌟 Components & Features Explored

This repository contains multiple standalone components, each focusing on a specific Angular concept:

*   **🚦 Signals (`/signal`)**: Demonstrates modern reactivity in Angular using `signal`, `WritableSignal`, and `effect` for state management without standard ChangeDetection overhead.
*   **🔢 Counter (`/counter`)**: A basic counter component showcasing state updates and template bindings.
*   **🖱️ Events (`/events`)**: Examples of event binding and handling user interactions.
*   **🔐 Login (`/login`)**: A practice implementation of a login form, likely exploring form control and input handling.
*   **🔁 Loops (`/loops`)**: Demonstrates traversing arrays and rendering lists (e.g., using Angular's `@for` control flow).
*   **👤 Profile (`/profile`)**: A user profile display component practicing data interpolation and layout.

## 🛠️ Tech Stack

*   **Framework**: [Angular](https://angular.dev/) (Standalone Components)
*   **Language**: [TypeScript](https://www.typescriptlang.org/)
*   **Styling**: CSS / HTML
